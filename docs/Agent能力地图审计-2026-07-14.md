## TikTok 采集边界

当前已部署：

- TikTok oEmbed 官方元数据读取。
- 人工 URL 与结构化元数据导入。
- 人工转写文本作为 Research Agent 输入。
- 参考内容仅用于结构、节奏、钩子形式和受众洞察，不复制竞品文案、声明、品牌或产品外观。

当前未部署：

- 经授权的 TikTok 视频文件下载。
- 下载后自动 ASR 转写。
- 视频抽帧、镜头切分、字幕/OCR 与音乐节奏检测。
- 点赞、评论、分享等发布表现的持续同步。

## 下一批建设优先级

1. Collector 增强：授权下载适配器、ASR、抽帧和镜头切分，输出可追溯的 `source_video_analysis`。
2. Asset 增强：为每镜生成场景式 contact sheet，而不是重复白底身份图。
3. Review 增强：人物一致性、产品结构、屏显温标和 Logo 漂移的视觉审核。
4. Feedback 增强：接入发布表现数据，形成策略优先级和模板版本更新建议。
5. Worker 对齐：将目前由 Engine 执行的能力迁移到真正可横向扩展的 Agent Worker handler。
