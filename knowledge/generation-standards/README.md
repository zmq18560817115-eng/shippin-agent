# 生成与合成标准

本目录把 OpenMontage 中适合本项目的通用做法转化为本地规则，不复制其供应商实现和密钥管理。

## 视频提示词

- 每个镜头必须给出主体、场景、动作、镜头运动、光线、时长和禁止项。
- 便携恒温杯外观只能来自已批准白底主图；禁止让模型自行推断结构、接口或配件。
- 明确 `vertical 9:16`、连续场景、角色外观和产品位置；动作使用可执行动词。
- 竞品素材只提供结构、节奏、钩子和受众洞察，不复刻台词、商标、宣称或视觉身份。

## 尺寸与导出

- 主交付：1080x1920、9:16、30fps、H.264、AAC、`yuv420p`。
- 输入镜头按比例缩放并居中补边，禁止拉伸产品。
- 每个输入镜头先标准化再拼接，避免分辨率、帧率和编码不一致导致无损拼接失败。

## 异步生成

- 保存 provider、model、task_id、request_id、开始时间、耗时和错误摘要。
- 轮询必须有总超时、指数退避和明确终态；下载失败可重试，生成失败只重试对应镜头。
- 供应商任务成功但本地文件为空视为失败，不进入合成。

## 最终 QA

- 必须读取真实探测结果，不允许静态 PASS。
- 强制检查 1080x1920、有效时长、24-60fps、至少一个音轨、镜头数一致、全部镜头成功、输出文件非空。
- 任一硬检查失败则 `BLOCKED`，不得生成发布归档。
